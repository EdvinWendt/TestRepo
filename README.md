# ReceiptScannerToSwish

Android app for scanning receipts, splitting totals between participants, and preparing Swish payment reminders.

## Website

The GitHub Pages website lives in `docs/`.

To publish it:

1. Push this project to the `TestRepo` repository.
2. Open `Settings` > `Pages` in GitHub.
3. Set the source to `Deploy from a branch`.
4. Select branch `main` and folder `/docs`, then save.

GitHub will publish it at:

`https://edvinwendt.github.io/TestRepo/`

If the repository is renamed to `ReceiptScannerToSwish` later, the URL will become:

`https://edvinwendt.github.io/ReceiptScannerToSwish/`
